Задание №3

Используя HTML, CSS и JavaScript сделать страницу с полем для ввода и блоком для вывода текста.

В поле для ввода пользователь может ввести дату в следующем формате: «месяц.день.год часы:минуты».

При вводе даты в поле для ввода она должна преобразовываться в другой формат: «часы- минуты день месяц год», причем месяц должен быть записан английским словом, например «January».

Выходная дата в блоке вывода должна обновляться автоматически с изменением содержимого поля для ввода.

Предполагается, что пользователь вводит только правильный формат входных дат, то есть проверку правильности формата даты делать не нужно.
